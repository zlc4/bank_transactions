PROJECT: Bank Database
DESCRIPTION: 

A dual‑engine ETL (Extract, Transform, Load) workflow for processing bank transaction data using either Pandas or PySpark. This project demonstrates how the same ETL logic can be implemented with two different data‑processing frameworks, allowing flexibility for both local and distributed workloads.

The load part of ETL takes place in SQL with the files provided. You will need to use the filepath from the ETL output file to use for Bulk Insert
